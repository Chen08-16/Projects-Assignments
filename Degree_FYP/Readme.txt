Prerequisites
Node.js: Ensure you have Node.js installed. You can download it from 
https://nodejs.org/en

MongoDB Server: Install MongoDB server. You can find the installation from https://www.mongodb.com/try/download/community


How to run project?
=============================================================================
Step 1: Install Dependencies
Open two terminal windows or tabs, one for the frontend and one for the backend.

In the frontend directory, run:
npm install
npm run dev


In the backend directory, run:
npm install
npm run dev
